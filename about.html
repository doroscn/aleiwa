About Aleiwa

What is Aleiwa?

Aleiwa is a network analytics and reference platform focused on public DNS servers, Autonomous System (ASN) rankings, and country-level internet infrastructure data.

The project was created to help network engineers, CDN operators, and infrastructure planners better understand how internet traffic is distributed across different regions and providers.

What We Provide

Aleiwa aggregates and organizes publicly available network data to present:

  • Country-level ASN rankings based on observed internet population distribution
  • Public and ISP-operated DNS server availability by region
  • Infrastructure insights useful for DNS selection, traffic routing, and CDN deployment planning

All data is presented in a human-readable format, making complex network information easier to explore and compare.

Data Sources & Methodology

The data shown on Aleiwa is collected from multiple publicly accessible sources, including:

  • BGP and ASN information from regional internet registries (RIRs) including ARIN, RIPE, APNIC, LACNIC, and AFRINIC
  • Public DNS datasets and active DNS availability checks
  • Network measurement data and reachability validation
  • IP-to-AS mapping datasets from various network research organizations

Data is processed, normalized, and updated on a regular basis to reflect changes in network topology and service availability. The platform does not rely on user-submitted data, ensuring objectivity and reliability.

Our methodology involves:

  • Regular collection of BGP routing table snapshots
  • Active DNS server testing and validation
  • Statistical analysis of IP address distribution by country and ASN
  • Cross-referencing multiple data sources for accuracy
  • Automated quality checks and data validation processes

Update Frequency

Aleiwa is actively maintained with regular data updates:

  • ASN rankings: Updated monthly to reflect changes in network infrastructure
  • DNS server data: Validated and updated weekly
  • Country-level statistics: Refreshed monthly with the latest routing information

Datasets and statistics are periodically refreshed to ensure accuracy and relevance as internet infrastructure evolves.

Who This Site Is For

Aleiwa is designed primarily for:

  • Network and systems engineers who need to understand regional network topology
  • CDN and DNS operators planning infrastructure deployment
  • Developers working on latency-sensitive applications
  • Researchers analyzing regional internet infrastructure
  • IT professionals making decisions about DNS selection and network routing

While the data is technical in nature, the site aims to make it accessible for anyone interested in how the global internet is structured.

How to Use Aleiwa

Our platform offers several tools and resources:

  • ASN Rankings: Browse country-specific ASN rankings to understand which network operators dominate in different regions
  • DNS Server Lists: Find the best public DNS servers for your region, with performance and reliability information
  • ISP Information: Explore top ISPs by country with detailed statistics
  • Blog Articles: Read our technical articles about network infrastructure, DNS optimization, and internet routing
